Welcome on Bixi Average Bikes Availability

Select a Day of the Week and an Hour and Click on the marker to get the average availability of bikes in your preferred station.

Information

The data was parsed during 2 months this summer (2019) using Bixi API every 5 seconds. This project is for education purpose only. Using Django (Python) / VueJS.

Bixi API
